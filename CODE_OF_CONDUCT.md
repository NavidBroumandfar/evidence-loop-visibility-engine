# Code of Conduct

We are committed to a welcoming, respectful, and evidence-minded community.
Participants must be constructive, assume good faith, protect private
information, and accept technical review without personal attacks.

Harassment, discrimination, doxxing, threats, and deliberate publication of
secrets are not acceptable. For sensitive conduct or security reports, use the
repository Security advisory private channel. If that channel is unavailable,
open a minimal public issue requesting private contact without personal details
or report contents.

For a security vulnerability, use the private advisory form:
https://github.com/NavidBroumandfar/evidence-loop-visibility-engine/security/advisories/new
