# Security policy

Please do not report secrets or private data in a public issue. For a suspected
vulnerability, use GitHub's private advisory form:
https://github.com/NavidBroumandfar/evidence-loop-visibility-engine/security/advisories/new
Include only a minimal reproduction, affected version, and safe mitigation
suggestion.

The package is intentionally offline and has no credential or provider access.
Keep any deployment or integration wrapper outside this repository and apply
its own review, isolation, and approval controls.
